üö® Critical Gaps Summary

Gap #1: Payment Capture Inconsistency

Guest orders are captured immediately, patron orders are authorized first. This creates two different code paths and inconsistent user experience.

Fix: Unify to one strategy (recommend immediate capture)

Effort: 2-3 days

Gap #2: Stripe Connect Payouts Not Operational

Payout cron job simulates payouts but doesn't execute actual bank transfers. Artisans see wallet balance but can't access funds.

Fix: Complete Stripe Connect integration

Effort: 1-2 weeks

Gap #3: No Artisan Onboarding UI

Backend endpoints exist for Stripe Connect setup, but no frontend interface. Artisans can't easily connect their bank accounts.

Fix: Build onboarding component

Effort: 3 days
